# Smart Task Organizer

Software Engineering Lab Project  
Palestine Polytechnic University

## Project Description
Smart Task Organizer is a simple application that helps users manage daily tasks.
It supports creating, editing, deleting, sorting, filtering, and exporting tasks.

## Features
- Add, edit, delete tasks
- Mark tasks as completed
- Sort tasks by deadline and priority
- Filter completed, not completed, and high priority tasks
- Save tasks locally using JSON
- Export tasks to a text file
- REST API with CRUD operations

## Technologies Used
- Python
- Flask (REST API)
- JSON (Local Storage)
